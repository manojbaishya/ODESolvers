## Available algorithms:

### Fixed stepsize
- Euler Forward
- Heun Predictor-Corrector
- Midpoint
- RK2 (Ralston-Rabinowitz version)
- RK3 (Simple and Optimal versions)
- RK4
- RK5 (Butcher)

### Variable stepsize
- Runge-Kutta-Fehlberg (Cash-Karp)
